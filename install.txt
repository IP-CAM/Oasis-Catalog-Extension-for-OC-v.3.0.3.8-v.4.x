-------
Установка
-------

- Скачайте архив «oasiscatalog.ocmod.zip» из последнего релиза

- Перейдите в раздел Модули -> Установка расширений, загрузите расширение «oasiscatalog.ocmod.zip» и установите модуль «Oasiscatalog»

- Перейдите в раздел Модули -> Модули (Extensions -> Extensions) и установите модуль Oasiscatalog.

- В настройках модуля Oasiscatalog введите API ключ и ID пользователя из личного кабинета https://www.oasiscatalog.com/cabinet/integrations и другие настройки при необходимости

- В панели управления хостингом добавить cron задачи со страницы настроек модуля
