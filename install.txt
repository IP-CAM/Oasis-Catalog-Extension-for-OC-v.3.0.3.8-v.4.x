-------
Установка
-------
- Для установки модуля через установщик расширений необходимо предварительно установить модуль Localcopy - https://ocmod.shop/localcopy

- Скачайте архив «oasiscatalog.ocmod.zip» из ветки https://github.com/oasis-catalog/opencart/tree/v.3.x

- Перейдите в раздел Модули -> Установка расширений и загрузите расширение oasiscatalog.ocmod.zip.

- Перейдите в раздел Модули -> Модули (Extensions -> Extensions) и установите модуль Oasiscatalog.

- В настройках модуля Oasiscatalog введите API ключ и ID пользователя из личного кабинета https://www.oasiscatalog.com/cabinet/integrations

- Во вкладке «Импорт» установите необходимые опции импорта и сохраните настройки модуля, после этого можете приступать к импорту.
