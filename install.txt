-------
Установка
-------

1. Перейдите в раздел Модули -> Установка расширений и загрузите расширение oasiscatalog.ocmod.zip.

2. Перейдите в раздел Модули -> Модули (Extensions -> Extensions) и установите модуль Oasiscatalog.

3. В настройках модуля Oasiscatalog введите API ключ и ID пользователя из личного кабинета https://www.oasiscatalog.com/cabinet/integrations

4. Во вкладке «Импорт» установите необходимые опции импорта и сохраните настройки модуля, после этого можете приступать к импорту.
